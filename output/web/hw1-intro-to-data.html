    Section 1.1 Homework 1

    Subsection 1.1.1 Terminology

    The following terminology will be used throughout the course:
    🔗

    Definition 1.1.1. Population.

    A collection of individuals (not necessarily people) that share one or more common characteristics.
    🔗
    🔗
    In statistics, this is the entire group of individuals for which information is being gathered or studied. Information that describes the entire population is called descriptive statistics
    🔗

    Definition 1.1.2. Sample.

    A subset of the population.
    🔗
    🔗
    In statistics, it is often neither feasible nor desirable to gather information about the entire population, so information is gathered on part of the population being studied. Inferential statistics describes when information gathered about the sample is used to say something about the population.
    🔗

    Example 1.1.3.

    Consider the total viewers of a television show. The following are examples of samples of this population: viewers in the United States; viewers between the ages 18 and 35; male-identifying viewers.
    🔗
    🔗

    Definition 1.1.4. Variable.

    A quantity or quality of the population that varies among individuals.
    🔗
    A quantitative variable is one which can be measured and which produces a numerical value. A qualitative variable or categorical variable is one which produces a description of the population.
    🔗
    Data are the values of variables.
    🔗
    🔗

    Note 1.1.5.

    Categorical variables can sometimes have numerical representations, if they are used to categorize individuals in the population.
    🔗
    🔗

    Definition 1.1.6. Parameter.

    A value found by looking at the entire population.
    🔗
    🔗

    Definition 1.1.7. Statistic.

    A value found by looking at a sample of the population.
    🔗
    🔗

    Example 1.1.8.

    In a recent survey, 1000 Oklahomans were polled on their opinion about a federal law requiring background checks for all gun purchases. 88% would support the law.
    🔗
    In this example, 88% represents a statistic because the 1000 people were chosen from the population of Oklahoma; it’s not feasible to ask the entire population, so a subset (1000 people) were asked.
    🔗
    🔗

    Example 1.1.9.

    A construction foreman asked his workers about their preferences for lunch. 45% wanted pizza, 20% wanted Mexican, 25% wanted Italian, and 10% wanted salad.
    🔗
    In this example, each percentage represents a parameter, not a statistic. As opposed to Example 1.1.8, it is possible to get information from all individuals on the worksite; this is why the values represent parameters rather than statistics.
    🔗
    🔗

      🔗

      🔗
      🔗

    Subsection 1.1.2 Types of Variables

    Example 1.1.11.

    Recall that Definition 1.1.4 introduced quantitative and qualitative variables (also called categorical variables).
    🔗
    Examples of quantitative variables describing OCCC students are: height, weight, age, GPA, and hours taken
    🔗
    Examples of qualitative variables describing OCCC students are: gender, race, classification (e.g. freshman, sophomore), hair color, phone number, ZIP code, and letter grade in a course.
    🔗
    🔗

    Note 1.1.12.

    Even though phone number and ZIP code display numbers, they are not numerical: they are classifying students by an identifier. These numbers are not measurements and cannot be used in further calculations (does the average of a phone number make sense?)
    🔗
    🔗

    Definition 1.1.13. Discrete/Continuous Variable.

    A quantitative variable is a discrete variable if it cannot take on any value between the minimum and maximum value of the variable being studied. The number of values that can be taken by the variable is clearly defined or can be counted.
    🔗
    A quantitative variable is a continuous variable if it can take on any value between the minimum and maximum value of the variable being studied.
    🔗
    🔗

    Example 1.1.14.

    The following are examples of discrete variables: age, shoe size, number of siblings, number of gas stations in a city.
    🔗
    The following are examples of continuous variables: height, weight, GPA, time, volume, pressure
    🔗
    🔗

    Check Your Understanding 1.1.15.

    Is the number of credit hours taken by an OCCC student this semester a discrete or continuous variable?
    🔗
    Answer.
    It is a discrete variable because it’s impossible to take anything other than a whole number of hours; taking 13.25 hours is impossible. Since you cannot take on any value between the minimum and maximum (0 and 21 hours), this is a discrete variable.
    🔗
    🔗
    🔗

    Distinguishing Between Continuous and Discrete Variables.

    Continuous variables are types of variables that we can get more and more precise measurements from. The precision is only limited by the device we are using to measure it. For example, someone’s weight could be 182.126 pounds and that number makes perfect sense. And if we had a better scale, we could get an even more precise measurement. For a discrete variable, the variable can only take on a certain number of values- not all values would make sense- like shoe size. 12.5 would make sense but a value like 12.456 would not. There is only a discrete (limited/countable) number of values that a discrete variable could have. Things that we are measuring scientifically tend to be continuous variables because the precision of measurement is only determined by the instrument doing the measuring.
    🔗
    🔗

    Definition 1.1.16. Nominal/Ordinal Variable.

    A qualitative variable is a nominal variable if there is no natural ordering of the variables.
    🔗
    A qualitative variable is an ordinal variable if there is a natural ordering of the variables.
    🔗
    🔗

    Example 1.1.17.

    Examples of nominal variables include: gender, race, hair color, ZIP code
    🔗
    Examples of ordinal variables include: year classification (e.g. freshman, sophomore), military rank, letter grade in a course.
    🔗
    🔗

    Subsection 1.1.3 Sampling Methods

    Most populations we study or work with within Statistics are of enormous size: think of the population of all U.S. households or the population of all eligible voters in the U.S. These populations contains tens of millions of individual data points, so it is not feasible (in terms of time, cost, etc.) to study the whole population directly. For populations where it is not desirable to study each individual from a population, we choose a representative sample.
    🔗

    List of Valid Sampling Methods.

    • Simple Random Sampling: We randomly choose the desired number of individuals out of the entire population of individuals. This method is valid because it allows all individuals an equal chance to be picked as part of a sample.
      🔗
      Simple random sampling is the most basic and common method for choosing a valid statistical sample out of a population
      🔗
      🔗
    • Stratified Random Sampling: This is a more advanced version of random sampling. We still choose a random sample; however, we also stratify it to closely match with the demographics of the population we choose our sample from. All medical and healthcare-related studies are conducted using this sampling method.
      🔗
      🔗
    • Cluster Sampling: Cluster sampling occurs when there are ’natural’ groupings evident in a population. It is often used in marketing research. In this technique, the total population is divided into these groups (called clusters) and a sample of the groups is selected.
      🔗
      🔗
    • Systematic Sampling: Sample members from a larger population are selected according to a random starting point and a fixed, periodic interval. This is often used when doing quality control for a particular product.
      🔗
      🔗
    🔗
    🔗

    Example 1.1.19.

    • Simple Random Sampling: Drawing names from a hat is an example of simple random sampling - everyone from the population is in the hat and has an equal opportunity to be chosen for the sample.
      🔗
      🔗
    • Stratified Random Sampling: Suppose we want to poll a sample of eligible voters to find out how they plan to vote in an upcoming election. In this population, 44% of voters are registered as Democrat, 42% as Republican, and 12% as independent. The stratified random sample we use for this poll should also reflect these percentages, so around 42% of all individuals should be chosen from the pool of all eligible Republican voters, 44% from all Democratic voters, and 12% from all independent voters.
      🔗
      🔗
    • Cluster Sampling: Suppose that Major League Baseball decides to do random drug testing during the season. They can do a cluster sampling by considering each division as one large cluster, and randomly choosing one or more divisions. Suppose the division ’AL East’ is chosen; this means that all members of the AL East (Yankees, Orioles, Red Sox, Rays) would be randomly tested.
      🔗
      🔗
    • Systematic Sampling: A soda pop company wants to make sure they have a consistent product so they systematically take every 100th can off the assembly line to get tested for consistency.
      🔗
      🔗
    🔗
    🔗

    List of Invalid Sampling Methods.

    • Voluntary-response Sampling: This occurs when the members of the sample choose to be part of the sample.
      🔗
      🔗
    • Non-response Sampling: This occurs when a sample is not large enough to represent a population.
      🔗
      🔗
    • Sample of Convenience: This occurs when the sample is chosen from convenient data points to the researcher.
      🔗
      🔗
    🔗
    🔗

    Example 1.1.22.

    A radio talk show in Oklahoma City asks people to chime in on who they think the best point guard in the NBA is.
    🔗
    • Voluntary-response Sampling: The people that call in will feel very passionate about whom they feel is the best and the fact they are listening to a particular radio station might indicate that they will all have similar views. It is very unlikely that this small sample calling into the radio station will feel the same way as the general population unless everyone in Oklahoma City listens to this radio station and they are all passionate about basketball.
      🔗
      🔗
    • Non-reponse Sampling: If 5 people call in to the station and all say that Shai Gilgeous-Alexander is the best point guard, this is not as convincing as if 500 people called in and declared Shai the best point guard.
      🔗
      🔗
    • Sample of Convenience: If the radio station claims that Shai Gilgeous-Alexander is the best point guard in the NBA based on the views of its callers, this is a sample of convenience because the sample (conveniently drawn from nearby listeners) is not representative of the entire population of NBA fans.
      🔗
      🔗
    🔗
    🔗

    Example 1.1.23.

    Sometimes a sample of convenience is ok.
    🔗
    A farmer wants to sell his newly harvested apples to a grocery store. In the back of his truck the farmer has his apples in numerous crates. The grocery store manager only examines the apples in the top of each crate (whether than checking the bottom) when deciding whether or not to buy the apples. A sample of convenience is ok when it can be used to represent the entire population.
    🔗
    🔗

    Subsection 1.1.4 Observational Studies and Experiments

    Definition 1.1.24. Statistical Studies.

    A statistical study is the way that statisticians and researchers gather information about the individuals from a population.
    🔗
    Studies can either take the design of an observational study or an experiment depending on the role of the researcher, the person conducting the study.
    🔗
    🔗

    Observational Study.

    In an observational study, the researcher only gathers information about a population through observation.
    🔗
    If the purpose of the observational study is to compare groups within the population, the researcher will not assign individuals from the population into the groups. The individuals either naturally fall within the groups or they have chosen to be in a group for themselves.
    🔗
    In an observational study the researcher does not manipulate or influence the behaviors of the population in any way.
    🔗
    🔗

    Example 1.1.25.

    Observational studies have linked smoking to poorer health outcomes. Researchers would follow smokers and non-smokers and record the health differences between the two groups.
    🔗
    This is an observational study because at no point did the researcher influence the behaviors of the individuals that they were observing.
    🔗
    Because of this lack of control for behaviors such as diet and exercise, researchers could only conclude that it was very likely that smoking had an adverse effect on one’s health. Also the researchers could not say that smoking was the direct cause of bad health.
    🔗
    🔗

    Note 1.1.26.

    Observational studies can usually show correlation (variables are related in some way) but not causation, where one variable is the direct cause of another (e.g. smoking is the direct cause of bad health).
    🔗
    🔗

    Experiments.

    In an experiment, the researcher manipulates the population in some manner.
    🔗
    One way that the researcher does this is by assigning individuals from the population into groups: experiment, which receives the experimental treatment, and control, which does not receive the experimental treatment.
    🔗
    Individuals do not choose for themselves which group they will be a part of. The researcher does this so that they can determine if administrating a treatment to one of the groups will cause a response that does not happen in the group that did not receive the treatment.
    🔗
    Unlike an observational study, the researcher in an experiment can influence the behaviors of subjects to control for lurking variables.
    🔗
    🔗

    Example 1.1.27.

    Consider a hypothetical experiment designed to determine whether or not smoking causes bad health.
    🔗
    For the experiment group, researchers randomly choose a sample of people from the population and give them a treatment. In this experiment, the individuals or subjects in this group would be asked to smoke (which would be highly unethical).
    🔗
    For the control group, researchers would randomly choose a sample of people from the population that will be told not to smoke.
    🔗
    Researchers would then try to control for lurking variables by controlling the behaviors of the two groups. For example, they might give both groups the same diet, exercise plan, require the same amount of sleep, etc.
    🔗
    In some experiments, researchers would also control for the placebo effect
    🔗
    🔗

    Definition 1.1.28. Placebo Effect.

    The placebo effect occurs when people think they are receiving a treatment when in actuality they are not. The mere thought of receiving the treatment in some individuals is enough to cause an effect.
    🔗
    🔗

    Terminology Associated with an Experiment.

    • Sample: If not specified it always means a simple random sample of individuals chosen from the population.
      🔗
      🔗
    • Subject: An individual chosen at random from a population for an experiment.
      🔗
      🔗
    • Explanatory Variable: A quantity used to measure the outcome of an experiment.
      🔗
      🔗
    • Response Variable: A quantity used to measure the outcome of an experiment.
      🔗
      🔗
    • Treatment: A combination of one or more explanatory factors in an experiment.
      🔗
      🔗
    • Double-Blind Experiment: An experiment where neither the researcher nor the subjects know which group (control or experimental) they are in.
      🔗
      🔗
    • Lurking Variables: Variables that have an influence on the response variable that are not accounted for in the study.
      🔗
      🔗
    🔗
    🔗

    Check Your Understanding 1.1.30.

    A fitness instructor wants to determine the effect that fitness classes have on body fat composition. Determine which is an observational study and which is a randomized experiment.
    🔗
    1. The fitness instructor measures the body fat composition of the participants of the 4 different classes that have already begun: yoga, Zumba, pilates, and Tae Bo. She later takes the body fat composition of each participant the last week of class to see which class made the largest improvement.
      🔗
      🔗
    2. She measures the body fat composition of 100 fitness class participants and then she randomly assigns the participants to yoga, Zumba, pilates, and Tae Bo. She later takes the body fat composition of each participant the last week of class to see which class made the largest improvement.
      🔗
      🔗
    🔗
    Answer.
    1. This is an observational study because the classes have already started. This implies that the participants chose for themselves which class(es) they would be in. The fitness instructor is not assigning treatment to the participants, which is essential for a randomized experiments.
      🔗
      🔗
    2. This is a randomized experiment because the fitness instructor is randomly assigning the treatment to participants.
